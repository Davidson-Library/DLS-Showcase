The Digital Learning & Scholarship team can help you transform quantitative and qualitative data into visually effective charts, graphs, and infographics. We can help you work with data using tools such as:

  • RAWGraphs: open-source tool that allows users to experiment with a wide range a chart types
  • Voyant Tools: web-based text analysis environment
  • Google Sheets / Excel: spreadsheet programs with data visualization tools
  • Canva: graphic design platform with basic data visualization tools
